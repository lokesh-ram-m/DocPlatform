# Technology Stack

| Concern                  | Technology                 | Details                          |
|------------------------- |--------------------------- |---------------------------------|
| Runtime & Frameworks     | .NET (net9.0)             | The application is built on .NET 9.0. |
| Runtime & Frameworks     | ASP.NET Core               | Used for building the web API.  |
| Frontend Framework        | Angular                    | The application utilizes Angular for the frontend. |
| Data Access              | Dapper                     | A micro-ORM used for data access. |
| Database                 | SQL Server                 | The application uses SQL Server for data persistence. |
| Authentication           | JWT Authentication         | Implemented for secure user authentication using JWT. |
| API Documentation        | OpenAPI                    | Provides a standard for API documentation. |
| API Documentation        | Swagger / Swashbuckle      | Tools for generating interactive API documentation. |
| AI / LLM                | Semantic Kernel            | Used for integrating AI capabilities within the application. |
| Security                 | BCrypt password hashing     | Used for securely hashing user passwords. |

This table categorizes the technologies employed in the application by their respective concerns, highlighting how each technology contributes to the overall functionality and architecture of TaskFlow.
