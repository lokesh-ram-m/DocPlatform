# Technology Stack

| Concern               | Technology                                      | Details                                                |
|----------------------|-------------------------------------------------|--------------------------------------------------------|
| Runtime & Frameworks  | .NET                                            | Target frameworks: net10.0 (various platforms)         |
|                      | ASP.NET Core                                    | Web framework for building APIs and web applications   |
|                      | EF Core                                         | Object-relational mapper for .NET                      |
| Data Access          | Entity Framework Core                           | Used for database operations                            |
|                      | PostgreSQL                                      | Database system utilized by the application             |
| Authentication       | ASP.NET Core Identity                           | Framework for user identity and authentication         |
|                      | JWT Authentication                              | Used to secure APIs with token-based authentication     |
|                      | OpenID Connect                                  | Provides an external identity solution                  |
| API Documentation    | OpenAPI                                         | Used for documenting APIs with Swagger                  |
| AI / LLM            | Azure AI                                        | Integration of Azure's AI capabilities                  |
|                      | OpenAI / GitHub Models                          | Utilizes OpenAI models for various functionalities      |
| Caching              | Redis                                           | Caching solution to enhance performance                 |
| Cloud                | Azure SDK                                       | Development toolkit for Azure cloud services            |
| Messaging            | RabbitMQ                                        | Messaging system used for communication among services  |
| Testing              | xUnit                                           | Testing framework for unit and integration tests        |
| Validation           | FluentValidation                                 | Library for validating user input                       |
