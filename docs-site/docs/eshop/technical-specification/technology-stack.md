# Technology Stack

| Concern                     | Technology                         | Details                                       |
|-----------------------------|------------------------------------|-----------------------------------------------|
| Runtime & Frameworks        | .NET                               | Targets net10.0 for various platforms (Android, iOS, MacCatalyst) |
|                             | ASP.NET Core                       | Web framework for building APIs and web apps |
|                             | EF Core                            | Object-Relational Mapper for data access     |
| Data Access                 | Entity Framework Core              | Provides database context and ORM features   |
|                             | PostgreSQL                         | Relational database for persistent storage    |
| Authentication              | ASP.NET Core Identity              | Framework for handling user authentication    |
|                             | JWT Authentication                 | Securely transmits information as a JSON object |
|                             | OpenID Connect                     | Protocol for authentication                   |
| API Documentation           | OpenAPI / Swagger                  | Framework for documenting APIs                |
| AI / LLM                   | Azure AI                           | Provides AI capabilities and services         |
|                             | OpenAI / GitHub Models             | AI model integration for enhanced functionalities |
| Caching                     | Redis                              | In-memory data structure store for caching    |
| Cross-cutting Libraries     | xUnit                              | Testing framework for .NET applications       |
|                             | FluentValidation                   | Library for model validation                  |
| Messaging                   | RabbitMQ                           | Message broker for asynchronous communication  |
|                             | EventBusRabbitMQ                  | Integration for handling events via RabbitMQ  |
|                             | IntegrationEventLogEF              | Service for logging integration events        |
|                             | EventBus                           | Provides an interface for publishing events   |
